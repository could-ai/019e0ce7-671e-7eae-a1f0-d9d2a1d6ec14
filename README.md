# Midnight Tales

A dark-themed, immersive horror story reading application built with Flutter.

## Overview
Midnight Tales provides a curated collection of terrifying short stories. The app features a dark, atmospheric UI designed for nighttime reading, with a focus on typography and immersive presentation.

## Features
*   **Dark Theme**: A custom deep red and black theme perfect for horror.
*   **Curated Stories**: A collection of bone-chilling short stories.
*   **Immersive Reading**: Clean, distraction-free reading view.
*   **Cross-Platform**: Works smoothly on mobile, web, and desktop.

## Tech Stack
*   Flutter
*   Dart

## Setup
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
